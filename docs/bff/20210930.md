# 菜市场 BFF 服务说明

## 开发指引
### 安装依赖
```bash
yarn install
```
### 配置环境变量
```bash
cp .env.example .env
```
### 服务启动和停止
```bash
# start server on production mode
yarn start
# start server on development mode
yarn dev

# stop server
# if run server in production mode an open daemon
yarn stop
```
